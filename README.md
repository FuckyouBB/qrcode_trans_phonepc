# qrcode_trans_phonepc
此工具帮助手机与电脑传输数据，在本地起server局域网通信

1.应用包生成二维码，手机扫码即可下载包，包括安卓和ios应用包

2.手机扫描工具中上传图片二维码，可将手机中图片传输到电脑上

3.电脑可传文本到手机，理论上，生成二维码可以传任意文件，所以可以把电脑上想传给手机的文字，放在txt文件

4.手机想传文本到电脑，可能还需要加代码
